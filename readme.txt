To show a currency widget (amount, from currency, to currency, result):
add the filter [[convert_currency]]

To add an inline converter to a page that will convert all prices on that poage to the selected currency, 
add the filter [[convert_currency_inline]] at the point you want the currency selector to appear
to set a default conversion currency for a particular page add the currency code on the end: [[convert_currency_inline:EUR]]
